# 21点小游戏 PRD

## 1. 项目概述

- **项目名称**：OpenClaw 21点
- **项目类型**：网页小游戏
- **核心功能**：经典21点纸牌游戏，支持单人vs电脑对战
- **目标用户**：休闲玩家

## 2. 游戏规则

### 基本规则
- 玩家和庄家各发2张牌
- 点数计算：A=1或11，2-10=面值，J/Q/K=10
- 目标：让手牌点数接近21且不超过21
- 超过21点爆牌，直接输掉游戏

### 操作流程
1. 点击"要牌"增加一张牌
2. 点击"停牌"保持当前点数
3. 庄家自动按照规则要牌
4. 比较点数大小决定胜负

### 计分规则
- 胜利：+10分
- 失败：-5分
- 平局：0分

## 4. 接口设计

### API 列表

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | /api/game/start | 开始新游戏 |
| POST | /api/game/hit | 要牌 |
| POST | /api/game/stand | 停牌 |
| GET | /api/game/result | 获取游戏结果 |
| GET | /api/score | 获取玩家分数 |
